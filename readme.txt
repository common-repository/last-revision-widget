Easily add a last revision date widget to your WordPress site sidebar.

== Description ==

Last Revision Date Widget adds a useful widget which you can use to see the last revision date of your post/page.

= Usage =

Simply go to `Appearance > Widgets` and drag "Last Revision Widget" to the sidebar of your choice.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Appearance > Widgets` and drag "Last revision date" to the sidebar of your choice.
